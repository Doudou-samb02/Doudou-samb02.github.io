📚 Gestion d’une Bibliothèque

Application web complète permettant la gestion efficace d’une bibliothèque, avec un système sécurisé pour les administrateurs et les lecteurs.

✨ Fonctionnalités principales

  • Gestion des utilisateurs (inscription, connexion, rôles)
  • Ajout, modification et suppression de livres
  • Gestion des auteurs et association aux livres
  • Emprunt et retour des livres avec suivi de disponibilité
  • Recherche et filtrage par titre, auteur et genre

🛠 Technologies utilisées

Spring Boot · Java · MySQL · HTML · CSS · JavaScript
